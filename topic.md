## Topic 06（推荐系统）动手做一个迷你推荐系统
Description 推荐系统广泛应用于手机软件和日常生活中，例如短视频平台会推荐用户可能喜欢的视频，音乐软件会推荐歌曲，购物软件会推荐商品，新闻 App 会推荐文章，学习平台会推荐课程或练习题。这些推荐并不是随机产生的，而是系统根据用户的点击、浏览、点赞、收藏、评分、购买、搜索等行为，推测用户可能感兴趣的内容。

本项目要求学生动手完成一个小型推荐实验，围绕“用户行为数据 → 用户—物品评分矩阵 → 相似度计算 → 推荐生成 → 结果解释 → 风险讨论”的思路，理解推荐系统的基本流程。学生建议优先使用MovieLens 等公开推荐系统数据集（https://grouplens.org/datasets/movielens/ ），选取其中一个小型子集进行实验；也可以自行收集同学对电影、音乐、图书、课程或食堂菜品的评分数据，作为贴近校园生活的拓展实验。本项目不要求实现复杂推荐算法，也不要求训练大型模型。重点是通过具体实验理解：推荐系统如何利用已有行为数据预测用户偏好，为什么相似用户可能会收到相似推荐，以及推荐系统在带来便利的同时，也可能造成数据稀疏、冷启动、信息茧房、隐私泄露和兴趣固化等问题。

可交付内容：推荐场景说明、数据来源说明、用户—物品评分矩阵、相似用户或相似物品分析、为至少2 位用户生成的推荐结果、推荐依据说明、成功推荐和错误推荐案例分析、推荐系统局限与风险讨论。

参考工具：scikit-learn cosine_similarity：用于计算余弦相似度；问卷星 / 腾讯问卷 / 飞书表单：可用于拓展组收集同学评分；手机软件案例：抖音、B 站、网易云音乐、豆瓣、淘宝、小红书、今日头条等。
